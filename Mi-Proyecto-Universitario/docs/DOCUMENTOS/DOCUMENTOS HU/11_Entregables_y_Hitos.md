# 11. Entregables y Hitos (Cronograma de Ejecución)

El desarrollo del proyecto se organizará en fases, permitiendo una implementación progresiva y controlada del sistema.

## 11.1. Fase 1: Análisis y Diseño

- Levantamiento de requerimientos del sistema.
- Análisis del proceso crediticio actual.
- Diseño de formularios y estructura del sistema.
- Definición de flujos de trabajo.

**Plazo estimado:** Hasta 40 días hábiles.


## 11.2. Fase 2: Desarrollo e Integración

- Desarrollo de módulos principales del sistema.
- Implementación de lógica de negocio.
- Integración con el CORE bancario mediante API REST.
- Implementación de funcionalidades clave (formularios, cálculos, reportes).

**Plazo estimado:** Hasta 140 días hábiles.


## 11.3. Fase 3: Pruebas y Validación (SandBox)

- Pruebas funcionales del sistema.
- Validación de cálculos financieros y productivos.
- Simulación de escenarios reales en entorno de pruebas.
- Capacitación inicial a usuarios.

**Plazo estimado:** Hasta 240 días hábiles.


## 11.4. Fase 4: Implementación y Puesta en Producción

- Despliegue del sistema en entorno productivo.
- Monitoreo inicial del funcionamiento.
- Ajustes finales.
- Entrega de documentación final y transferencia tecnológica.

**Plazo estimado:** Hasta 260 días hábiles.
