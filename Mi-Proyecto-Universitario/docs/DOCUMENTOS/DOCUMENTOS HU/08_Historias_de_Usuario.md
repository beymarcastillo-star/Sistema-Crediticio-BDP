## 8.4. Consideraciones

- Las historias están basadas en el enfoque ágil (Scrum), priorizando el valor para cada tipo de actor.
- Se incluyen actores operativos, de negocio y técnicos (DevOps), conforme a la guía de transformación IEEE → Ágil.
- Las historias técnicas (OVS) garantizan la calidad, seguridad e integración del sistema.
