\# 🚀 Proyecto: Digitalización de Procesos BDP



Repositorio centralizado para el desarrollo del Sistema Web y la documentación de las asignaturas de Ingeniería de Software y Gerencia de Proyectos.



\## 📁 Organización del Repositorio

\- `clases/ingenieria-de-software/`: Documentación técnica del proyecto BDP en formato Markdown.

\- `clases/gerencia-de-proyectos/`: Planes financieros y actas de constitución (Project Charters).

\- `sistema-web/`: Código fuente de la plataforma.

\- `assets/`: Imágenes de guía, diagramas y recursos visuales.
